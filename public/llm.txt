## Product Definition
IntegrateWise is the world's first Universal Cognitive Operating System — an AI-powered Knowledge Workspace that acts as a context-aware layer connecting everything a person already uses into one intelligent system.

Not just SaaS tools — the entire technology stack:
- Tools (Notion, HubSpot, Jira, Salesforce, Zendesk)
- AI Assistants (Claude, ChatGPT, Perplexity)
- Data Storage (databases, spreadsheets, CSVs)
- Charts and Analytics
- Documents and Context
- Institutional Memory

It does NOT replace any technology. It sits across all of it and makes every category aware of every other category.

## Architecture Summary
- Four Layers: L0 (Onboarding), L1 (15 Workspace Modules), L2 (14 Cognitive Components), L3 (Backend Engine)
- 8-Stage Mandatory Pipeline: Analyzer → Classifier → Filter → Refiner → Extractor → Validator → Sanity Scan → Sectorizer
- Three Data Flows: A (Structured Tools), B (Unstructured Docs), C (AI Chat via MCP)
- Engine Loop: LOAD → NORMALIZE → STORE → THINK → REVIEW & APPROVE (GOVERN) → ACT → REPEAT
- Three Spaces: Personal, Work, Team
- Progressive Hydration: B0 (Identity) through B7 (Full Intelligence)

## Agent Architecture
- 7 Domain Agents: SuccessPilot, ChurnShield, VaultGuard, DataSentinel, ArchitectIQ, TemplateForge, DealDesk
- Colony System: Multi-agent orchestration for complex tasks
- All agents are approval-first, evidence-backed, goal-linked
- No agent executes autonomously — all actions require Govern approval tokens
