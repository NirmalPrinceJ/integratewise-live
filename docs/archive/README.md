## 🔍 When to Reference Archive

Archive documents are useful for:
- **Historical context**: Understanding why certain decisions were made
- **Pattern mining**: Finding solutions to similar problems solved before
- **Onboarding**: Showing new team members the evolution of the system
- **Compliance**: Demonstrating due diligence and planning rigor

## ⚠️ Important Notes

- **Do not link** to archive docs from active documentation
- **Do not update** archive docs (they're historical snapshots)
- If you need to resurrect content, copy it to a new doc with a fresh perspective
- Archive docs retain their original dates and status markers
